输出格式化: 

def size_fmt(s, pos = 0):
    for f in 'BKMG':
        if s < 1024:
            break
        s = s / 1024
    else:
        f = 'T'

    return '%.1f%s' % (s, f)

  • 设置 X 轴的显示: plt.gca().xaxis.set_major_formatter(ticker.FuncFormatter(size_fmt))
  • 设置 X 轴的步长: plt.gca().xaxis.set_major_locator(ticker.MultipleLocator(1024 * 64))
